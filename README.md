# TeamSync 🚀

TeamSync is a full-stack project management workspace designed to streamline task tracking. Built with a focus on seamless user experience, it features a drag-and-drop Kanban board, real-time data visualization, and a fully responsive, mobile-first design.

## ✨ Key Features & Architecture
* **Monorepo Structure:** Cleanly separated `client` and `server` environments within a unified codebase.
* **Interactive Kanban Board:** Drag-and-drop task management powered by optimistic UI updates for zero-latency interactions.
* **Responsive Layout:** Custom horizontal-scrolling columns for mobile screens and a dynamic layout that adapts gracefully from desktop to phone using Tailwind CSS.
* **Project Analytics:** Integrated Doughnut charts for a quick visual overview of task statuses.
* **Serverless Backend:** RESTful API built with Express and deployed as a serverless function on Vercel for instant, on-demand scaling.

## 🛠️ Tech Stack
* **Frontend:** React.js, Vite, Tailwind CSS
* **Backend:** Node.js, Express.js 
* **Database:** MongoDB Atlas
* **State & Data Management:** Axios, React hooks
